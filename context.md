Este projeto é uma plataforma de compartilhamento de promoções de hardware e tecnologia. O objetivo é permitir que os usuários compartilhem promoções de hardware e tecnologia, votem nelas e comentem.

Ele roda em container sendo um para o frontend, um para o backend e um para o banco de dados.

O frontend é feito em React com TypeScript e Vite.
O backend é feito em Node.js com TypeScript e Express.
O banco de dados é feito em PostgreSQL.

O frontend se comunica com o backend através de uma API REST.
O backend se comunica com o banco de dados através de uma conexão direta.

O frontend é responsável por:
- Exibir as promoções
- Permitir que os usuários votem nas promoções
- Permitir que os usuários comentem nas promoções
- Permitir que os usuários criem promoções
- Permitir que os usuários se cadastrem e façam login
- Permitir que os usuários vejam seu perfil

O backend é responsável por:
- Armazenar as promoções
- Armazenar os votos
- Armazenar os comentários
- Armazenar os usuários
- Autenticar os usuários
- Autorizar os usuários
- Validar os dados
- Retornar os dados para o frontend 

O banco de dados é responsável por:
- Armazenar as promoções
- Armazenar os votos
- Armazenar os comentários
- Armazenar os usuários
- Autenticar os usuários
- Autorizar os usuários
- Validar os dados
- Retornar os dados para o frontend
